# TODO Index

## Description
This app is for user to challenge their coding skills by solving coding problems. everyday the question will shown and generated by AI.
- user need to choose the language / framework then select the level is it beginner, intermediate, advanced, expert. 
- user need to solve the question once every day.
- one session can have 5 questions.
- the question will show and change everyday 
- the question is multiple choice question with 4 options.
- user need to select the correct answer.
- system will take the time taken to solve the question, and it will calculate the score based on the time taken to solve the question.
- user will get score based on the question and the time taken to solve the question. user will get badge based on the score. 
- user will get ranking based on the score. user will get reward based on the score. 
- user will get achievement based on the score. user will get challenge based on the score. user will get leaderboard based on the score.
